String: Tipo de dato que representa texto.

Number: Tipo de dato para valores numéricos (enteros o decimales).

Union Type: Permite que una variable acepte más de un tipo de dato (ej: string | number).

Interface: Estructura que define la forma de un objeto (sus propiedades y tipos).

Enum: Conjunto de constantes con nombre, útil para valores fijos como niveles de error.
